## 配置 Ts 环境

我使用 webstorm 来作为开发工具。

```shell
node -v
npm install typescript -g
tsc --version
```

## 配置 WebStorm

`tsconfig.json` 配置相应配置。

右键编译或者使用设置中的 `FileWatcher` 自动编译。
